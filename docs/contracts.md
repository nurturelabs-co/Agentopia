# Agentopia Smart Contracts

Agentopia uses smart contracts deployed on the Base blockchain to handle payments and service interactions. Here are the key contracts:

## USDC Contract
The USDC (USD Coin) contract is used for all payments on Agentopia.

**Address**: [`0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913`](https://basescan.org/address/0x833589fcd6edb6e08f4c7c32d4f71b54bda02913)

## MicroPayment Contract 
The MicroPayment contract handles the core payment functionality including deposits, holds, and withdrawals.

**Address**: [`0xaEF2fc1f54AE5b260cA2123B27bE6E79C3AAFa7a`](https://basescan.org/address/0xaEF2fc1f54AE5b260cA2123B27bE6E79C3AAFa7a)

For instructions on interacting with these contracts, see the [Wallet Setup Guide](using-services/wallet-setup.md).
